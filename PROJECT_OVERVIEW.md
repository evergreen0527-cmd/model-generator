# 🎨 模特图生成器 - 项目概览

## 📋 项目简介

这是一个基于 AI 的模特图片生成 Web 应用,允许用户:
- 创建和管理多个模特档案
- 为每个模特上传参考素材图
- 上传场景参考图,生成模特在新场景中的图片
- 每个模特拥有独立的工作空间

## 🔌 API 接口

### 模特相关

| 方法 | 路径 | 说明 | 请求体 |
|------|------|------|--------|
| GET | `/api/models` | 获取所有模特 | - |
| POST | `/api/models` | 创建新模特 | FormData(name, referenceImage) |
| GET | `/api/models/:id` | 获取模特详情 | - |
| DELETE | `/api/models/:id` | 删除模特 | - |

### 图片生成

| 方法 | 路径 | 说明 | 请求体 |
|------|------|------|--------|
| POST | `/api/models/:id/generate` | 生成场景图 | FormData(sceneImage, prompt) |

## 🚀 启动流程

```
1. 配置环境变量
   └─> 复制 .env.example 为 .env
   └─> 填入 OpenAI API Key

2. 启动后端 (端口 3001)
   └─> npm install
   └─> npm run dev

3. 启动前端 (端口 3000)
   └─> npm install
   └─> npm run dev

4. 访问应用
   └─> 浏览器打开 http://localhost:3000
```

## 💡 使用流程

```
用户操作流程:

1. 创建模特
   ├─> 点击"创建新模特"
   ├─> 输入模特名称
   ├─> 上传模特参考图
   └─> 保存

2. 选择模特
   └─> 从左侧列表选择模特

3. 生成场景图
   ├─> 上传场景参考图
   ├─> (可选)输入提示词
   ├─> 点击"生成模特图"
   └─> 等待 AI 生成

4. 查看结果
   └─> 在页面底部查看生成的图片
```

## 🔧 关键技术点

### 前端
- **React Hooks**: useState, useEffect
- **组件化设计**: 模块化组件结构
- **状态管理**: 组件级状态管理
- **文件上传**: FormData API
- **图片预览**: URL.createObjectURL

### 后端
- **Express.js**: RESTful API 框架
- **Multer**: 文件上传中间件
- **OpenAI SDK**: API 调用封装
- **UUID**: 唯一 ID 生成
- **CORS**: 跨域资源共享

### 安全性
- 文件类型验证
- 文件大小限制(10MB)
- 唯一文件名生成
- 环境变量管理

## 📊 数据流

```
创建模特:
前端 ──FormData──> 后端 ──保存文件──> uploads/
                   └─创建记录──> 内存存储
                   └─返回数据──> 前端更新UI

生成图片:
前端 ──FormData──> 后端 ──保存场景图──> uploads/
                   └─调用 OpenAI API ──> 生成图片
                   └─保存记录──> 模特数据
                   └─返回结果──> 前端展示
```
