## 工作流程

### 1️⃣ 创建模特

```
用户操作:
1. 点击 "+ 创建新模特"
2. 填写模特名称
3. 上传清晰的模特照片
4. 点击"创建"

系统响应:
✓ 保存模特信息
✓ 生成唯一ID
✓ 存储参考图片
✓ 更新模特列表
```

### 2️⃣ 选择模特

```
用户操作:
1. 在左侧列表点击模特

系统响应:
✓ 加载模特详情
✓ 显示工作空间
✓ 展示已生成的图片
```

### 3️⃣ 生成场景图

```
用户操作:
1. 上传场景参考图
2. (可选)输入描述提示词
3. 点击"✨ 生成模特图"

系统响应:
✓ 上传场景图片
✓ 调用 AI 模型
✓ 生成新图片
✓ 更新展示列表
```

## 🎨 色彩主题

### 主色调
- **渐变紫**: #667eea → #764ba2
- 用于: 背景、按钮、强调元素

### 辅助色
- **白色**: #ffffff (卡片背景)
- **浅灰**: #f5f5f5 (悬停效果)
- **中灰**: #e0e0e0 (边框)
- **深灰**: #333333 (文字)

### 状态色
- **成功**: 绿色
- **错误**: 红色 (#fee 背景, #c33 文字)
- **加载**: 紫色旋转动画

## ⚡ 交互效果

### 按钮
- 默认: 渐变背景
- 悬停: 向上移动 2px + 阴影
- 禁用: 降低透明度

### 卡片
- 默认: 白色背景 + 边框
- 悬停: 变换边框颜色 + 背景色
- 激活: 高亮边框 + 浅色背景

### 上传区域
- 默认: 虚线边框
- 悬停: 实线边框 + 变色
- 有图片: 显示预览

### 加载状态
- 按钮内显示旋转动画
- 禁用按钮防止重复提交
- 文字提示"生成中..."

## 📊 数据流图

```
用户界面
   │
   ├─ 创建模特 ──> POST /api/models ──> 保存文件+数据 ──> 返回列表
   │
   ├─ 选择模特 ──> GET /api/models/:id ──> 获取详情 ──> 显示工作区
   │
   ├─ 生成图片 ──> POST /api/models/:id/generate ──> AI生成 ──> 更新列表
   │
   └─ 删除模特 ──> DELETE /api/models/:id ──> 删除数据 ──> 刷新列表
```
